# 表达高手微信小程序代码原型

这是一套原生微信小程序结构的代码原型，当前已经接入本地后端和 SQLite 数据层，重点用于验证 V1 的页面结构、交互主链和会员拦截逻辑。

主链路：

- 首页
- 4×4 卡片训练
- 30 秒思考
- 2 分钟表达
- AI 点评
- 历史记录
- 历史详情
- 我的 / 会员开通

另外附带一个最小后台原型：

- `admin/index.html`

## 原型特性

- 每轮固定 16 张卡，全部为形容词词卡
- 用户可自由选择任意 2 张卡进入训练，不限制词性组合
- 当前轮内卡片状态会在 `hidden / flipped / used` 之间流转
- 思考阶段包含 30 秒倒计时
- 表达阶段包含 2 分钟倒计时
- AI 点评根据会员状态展示精简版或完整版
- 免费用户默认 3 次训练轮次，会员不受限制
- 表达页采用“录音转写模拟”而不是纯文本输入
- 历史记录展示最近训练结果

## 目录结构

- `app.*`：应用级配置和后端调用封装
- `pages/*`：微信小程序页面
- `utils/api.js`：本地后端请求封装
- `admin/*`：最小后台网页原型

## 说明

- 当前已接本地 SQLite 数据库
- 会员开通、训练历史、后台列表都会真实写库或读库
- AI 评分默认走本地评分器；如果配置模型接口环境变量，可切换到真实模型调用
- 录音转写目前仍是前端生成文本片段，不是真实 ASR
- 如果手机扫码时访问不到你电脑本地后端，小程序会自动切换到内置演示模式，方便真机录屏
