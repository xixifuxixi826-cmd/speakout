# 表达高手 首批优先训练词 64

这 64 个词不是为了覆盖所有议题，而是为了先让产品前几轮训练的质量更稳。

筛选原则：

- 日常讨论频率高
- 词义清楚但有延展空间
- 容易形成 `A 是 B` / `A 是一种 B`
- 既能讲个人处境，也能讲社会关系
- 适合普通用户开口，不会太生僻

## 为什么先用这 64 个

这四组词最适合做首批正式训练，原因是：

1. 和普通用户的真实生活距离最近  
2. 既能讲个人感受，也能讲结构性问题  
3. 更容易产出“看似跳跃、其实能讲通”的判断句  
4. 对首轮评分系统最友好，方便我们先把标准校准

## 暂时放在第二批的方向

后续可以再逐步增强：

- 家庭与代际
- 女性与性别
- 消费与金钱
- 社交与边界
- 社会与价值
- 时代与选择
- 表达与说服
- 互联网与身份
- 修复与成熟

这些主题非常有价值，但如果一开始全部混进高频训练里，会让产品前几轮的调性不够稳定。
