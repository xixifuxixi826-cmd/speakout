## 图 1｜产品规划思维导图

```mermaid
mindmap
  root((开口 V1))
    产品定位
      微信小程序首发
      联想卡片表达训练
      不做标准答案教学
      强迫用户主动建立联系并开口表达
    核心价值
      训练关联能力
      训练观点组织能力
      训练即兴反应能力
      提供结构化 AI 反馈
      沉淀个人表达成长记录
    目标用户
      学生
      职场人
      内容创作者
      想提升表达能力的普通用户
      不喜欢被教学化的人群
    V1 产品目标
      快速开始一次训练
      用随机词组合制造表达压力
      用 AI 反馈指出问题与方向
      形成可回看的历史记录
    核心模块
      首页
        剩余次数
        开始训练入口
        最近一次评分
        会员状态
      卡片训练
        4x4矩阵
        8个名词
        8个形容词
        任意两词都可训练
        卡片状态
          未翻开
          已翻开
          已使用
      表达流程
        30秒构思
        2分钟表达
        语音优先
        文本兜底
      AI反馈
        评分
        问题点评
        改进建议
        改写示例
        免费版精简
        会员版完整
      历史记录
        训练结果沉淀
        支持回看详情
        支持成长感知
      会员体系
        免费用户每日3次
        会员无限训练
        会员解锁完整反馈
        会员解锁更长历史可见范围
      最小后台
        仪表盘
        用户列表
        订单列表
        词语管理
    关键机制
      任意两词都允许表达
      不校验词性搭配
      一轮用完16张卡才结束
      同一用户当天词语尽量不重复
      失败重点不在答错而在没讲清楚
    核心指标
      单次训练完成率
      次日留存
      平均每日训练次数
      会员转化率
    V1 边界
      不做社交互动
      不做多训练模式
      不做专业领域词库扩展
      不做完整语音评分
      不做复杂后台权限系统
    长期扩展
      AI辩论训练
      演讲训练
      语音分析
      语音播报
      难度分层
      动态词库生成
      更完整内容中心
```

## 图 2｜App 逻辑设计流程图

```mermaid
flowchart TD
    A[用户进入开口小程序] --> B[微信授权登录 / 绑定openid]
    B --> C[进入首页]

    C --> C1[查看今日剩余训练次数]
    C --> C2[查看会员状态]
    C --> C3[查看最近一次训练结果]
    C --> D{免费次数是否已用尽?}

    D -- 否 --> E[点击开始训练]
    D -- 是 --> M1[显示开通会员继续训练]
    M1 --> M2[进入会员开通页]
    M2 --> M3{是否开通会员?}
    M3 -- 是 --> E
    M3 -- 否 --> C

    E --> F[初始化一轮4x4卡片矩阵]
    F --> F1[生成16张卡]
    F1 --> F2[8个名词 + 8个形容词]
    F2 --> F3[过滤同用户当天已出现词语]
    F3 --> G[进入卡片页]

    G --> H[用户翻开卡片]
    H --> I[卡片状态: 未翻开 -> 已翻开]
    I --> J[用户自由选择2张卡]
    J --> K{是否正好选中2张?}
    K -- 否 --> J
    K -- 是 --> L[进入30秒构思阶段]

    L --> L1[快速建立观点]
    L1 --> N[进入2分钟表达阶段]
    N --> N1[语音输入优先]
    N --> N2[文本输入兜底]
    N1 --> O[提交分析]
    N2 --> O

    O --> P[系统调用AI分析]
    P --> Q{AI分析是否成功?}
    Q -- 否 --> Q1[保留本次提交内容]
    Q1 --> Q2[提示稍后重试分析]
    Q2 --> O
    Q -- 是 --> R[生成结构化AI反馈]

    R --> R1{用户是否为会员?}
    R1 -- 免费用户 --> S[展示精简版反馈]
    R1 -- 会员用户 --> T[展示完整版反馈]

    S --> U[更新本次2张卡为已使用]
    T --> U
    U --> V[保存训练记录]
    V --> W{当前16张卡是否全部已使用?}

    W -- 否 --> X[返回卡片页继续翻牌与选词]
    X --> H

    W -- 是 --> Y[提示本轮完成]
    Y --> Z{是否进入下一轮?}
    Z -- 否 --> C
    Z -- 是 --> Z1{当前是否仍可继续训练?}
    Z1 -- 可继续 --> F
    Z1 -- 免费额度已用尽 --> M1

    C --> H1[进入历史记录页]
    H1 --> H2[查看训练列表]
    H2 --> H3[查看单次训练详情]

    C --> P1[进入我的页面]
    P1 --> P2[查看会员状态]
    P1 --> P3[进入会员开通页]
```
